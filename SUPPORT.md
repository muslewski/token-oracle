# Support

Thanks for using **token-oracle**.

## Where to go

- **Questions & ideas** → [GitHub Discussions](https://github.com/muslewski/token-oracle/discussions)
- **Bugs & regressions** → [GitHub Issues](https://github.com/muslewski/token-oracle/issues/new/choose)
- **Security** → see [SECURITY.md](./SECURITY.md) (private report only)
- **Docs & install** → [README.md](./README.md) · website linked from the README

## Before you open an issue

1. Check existing issues and discussions for duplicates.
2. Confirm you're on the latest release (`token-oracle --version` / package version).
3. Include OS, version, and the exact command that failed.
