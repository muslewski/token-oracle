## Why this matters

token-oracle is a published PyPI package (v0.1.1, Alpha) whose docs are the product surface for adopters. Four docs actively lie about the code: README claims a TOML config and a stdin-based generic source (both wrong), and its color table describes thresholds that don't exist in the code; ADAPTERS.md and SETUP.md — the two documented extension paths — give Python examples that import a package name (`oracle.`) that was renamed to `token_oracle` before release, so every copy-paste example fails with `ModuleNotFoundError`; AGENTS.md is a deterministic agent runbook whose expected outputs no longer match what the CLI prints, so an agent following it faithfully will conclude the install is broken. Stale docs are worse than missing docs.

def gauge_tier(pct):
    """Severity tier for a projected pct. The one source of truth for thresholds."""

    if pct >= 85:

    return "green"
