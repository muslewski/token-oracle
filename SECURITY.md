# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email **10kento10@gmail.com** with subject line `[token-oracle] Security vulnerability`. Include:

- Description of the vulnerability and its potential impact
- Steps to reproduce
- Suggested fix (optional)

You will receive a response within 48 hours. Once confirmed and fixed, a security advisory will be published and the reporter credited (unless anonymity is preferred).
